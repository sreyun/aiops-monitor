功能详情

每个功能都为运维效率而生

不是技术特性的罗列，而是解决实际运维问题的业务能力

实时指标监控

CPU 使用率、内存/SWAP、多磁盘、网络收发速率、系统负载、进程数、TCP 连接数、运行时间 —— 5 秒级采集，全面覆盖。

GPU 监控：NVIDIA（nvidia-smi）、AMD（Linux sysfs）、Apple（macOS ioreg），best-effort + 缓存。多级降采样保留 7 天历史趋势。

告别逐台 SSH top/free/df 查看指标

自定义拨测监控

HTTP 状态码检测、TCP 端口连通性、Ping 延迟、关键进程存活 —— 四种拨测类型覆盖所有服务可用性场景。

内置历史曲线图，支持框选放大和全屏预览。

服务不可达第一时间发现

多渠道智能告警

CPU/内存/磁盘/负载/GPU/离线 六类阈值告警，分级严重/警告。飞书 Webhook、钉钉 Webhook（HMAC 签名）、SMTP 邮件三渠道推送。

事件去重冷却（5 分钟内相同事件不重复推送），桌面通知 + 声音提醒。

告警噪音降低 80%，只看需要处理的

远程终端（免开端口）

浏览器直接打开主机终端，Agent 反向连接服务端，无需在主机上开放任何入站端口。支持多标签页、窗口大小自适应。

完整 VT100 仿真器，支持 vim/top 等全屏程序。移动端虚拟键盘支持。

故障排查不用 VPN + SSH，浏览器直达

自动化剧本编排

可视化定义命令序列，一键批量执行到多台主机。执行结果实时回传，支持成功/失败状态统计和步骤级输出查看。

执行历史完整保留，操作者、时间、结果全部可追溯。

100 台主机批量打补丁，10 分钟完成

多用户 RBAC + MFA

三级角色权限控制：管理员（全部操作）、操作员（终端+告警）、观察员（只读查看）。路由级权限拦截，用户管理界面。

TOTP 两步验证（RFC 6238，Google Authenticator 兼容）。支持账户找回：忘记用户名/忘记密码（邮箱验证码）/ 邮箱解除 MFA，防枚举。

团队协作安全合规，满足等保要求

终端会话回放

所有远程终端会话全程录制（时间戳帧），支持 1x/2x/4x/8x 倍速回放。实时旁观功能让多人同时查看活跃会话。

会话列表支持按操作者用户名、主机名、IP 地址三维搜索。

谁做了什么，什么时候做的，完整可追溯

Python 插件扩展

内置插件 SDK，几行代码即可采集自定义指标（MySQL 连接数、Nginx 请求量、Redis 内存等）。

内置示例：进程监控、服务端口探活、轻量 AI 异常检测（z-score）。

监控什么你说了算，不限于内置指标

PWA 离线访问

支持安装到桌面（PWA），独立窗口运行。App Shell 离线缓存，断网时仍可查看最后已知状态。

WebSocket 实时推送 + 轮询降级，网络恢复自动重连。

手机也能装，随时随地查看监控

多服务端推送

单个 Agent 同时向多个服务端推送数据，采集一次广播所有。各服务端独立鉴权/重试，适合容灾或多团队共享监控。

一套 Agent，多套监控面板

网关中继模式

内网仅一台联网机器代理所有请求到云端，二进制/上报/终端自动穿透。适合跨网段或防火墙后的主机纳管。

无需每台机器都开外网

机器指纹鉴权

machine-id + MAC 哈希指纹绑定，Agent 终端通道按指纹鉴权（非 Token）。Token 轮换不影响已装 Agent，7 天宽限期。

Token 轮换不中断已部署 Agent

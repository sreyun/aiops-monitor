- HTTP 路由统一在 `Server.Routes()` 中通过 `mux.HandleFunc("METHOD /path", s.handleXxx)` 集中注册，并按功能域分组注释（如 SRE workflow、Playbooks、Terminal 等）。
- 平台差异通过 Go build tag 文件拆分（`*_linux.go` / `*_windows.go` / `*_darwin.go` / `*_other.go`），同一接口在不同 OS 提供实现。
- 对外部依赖（PostgreSQL、VictoriaMetrics）采用「必须可用」策略：启动时重试连接若干次，失败则直接 `log.Fatal` 退出，禁止静默降级到本地文件存储。
- 敏感配置（MFA/SMTP/AI/webhook/relay 密钥）通过 `AIOPS_SECRET_KEY` 经 AES-256-GCM 静态加密落库，未设置时仅记录 Warn 日志提示风险。
- 版本信息通过 `ldflags -X main.appVersion=...` 注入，并在 `main.go` 中以 `appVersion` 变量暴露给 dashboard 与 summary API。
- 响应压缩与安全头通过中间件链组合：`securityHeadersMiddleware → corsMiddleware → gzipMiddleware → bodyLimitMiddleware → authMiddleware`，WS 升级与 streaming 路径显式跳过压缩。

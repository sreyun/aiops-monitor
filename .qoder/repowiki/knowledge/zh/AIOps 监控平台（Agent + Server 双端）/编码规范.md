- 所有 Agent→Server 上报接口均通过 `X-Agent-Fingerprint` 请求头或 `fp` query 参数进行指纹校验，禁止在 body 中携带凭据。
- 配置项优先从 JSON 配置文件加载，再通过命令行 flag 覆盖（如 `-server`、`-interval`、`-config`），且首次启动自动生成 `config.example.json`。
- 跨平台差异通过文件名后缀区分（`*_linux.go`、`*_windows.go`、`*_darwin.go`、`*_other.go`），同一功能在不同 OS 下提供独立实现文件。
- 周期性任务统一使用 `time.NewTicker` + goroutine 模式，并在外层包裹 `defer recover()` 防止 panic 杀死整个进程。
- 对外 HTTP 响应一律通过统一的 `writeJSON` 辅助函数返回，错误路径直接返回带 `error` 字段的 JSON，避免手写 `json.Marshal`。
- 对第三方设备（BMC/iDRAC/iLO）的 TLS 握手单独放宽至 TLS 1.0 并启用不安全密码套件，同时关闭 KeepAlive，与普通 Agent→Server 的严格 TLS 1.2+ 策略隔离。
